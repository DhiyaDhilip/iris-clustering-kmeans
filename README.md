# iris-clustering-kmeans
Unsupervised learning project applying K-Means clustering on the Iris dataset. Includes data visualization, Elbow Method, Silhouette Score evaluation, and cluster interpretation using Python and Scikit-learn.

#  Iris Dataset Clustering Using K-Means

##  Project Overview
This project applies K-Means clustering on the Iris dataset to group flowers based on their features. The goal is to explore natural groupings using unsupervised learning techniques.

##  Objectives
- Apply clustering on structured data
- Determine optimal number of clusters
- Visualize and interpret cluster results

##  Tools & Technologies
- Python 3
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

##  Workflow
1. Data loading and preprocessing  
2. Feature scaling  
3. Exploratory Data Analysis (EDA)  
4. Elbow Method for optimal clusters  
5. Silhouette Score evaluation  
6. K-Means clustering  
7. Cluster visualization  

##  Key Insights
- Successfully identified natural clusters in the dataset
- Visualization confirmed meaningful separation between clusters
